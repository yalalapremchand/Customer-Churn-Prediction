# 📊 Customer Churn Prediction

A Machine Learning web application that predicts whether a customer is likely to churn based on customer information. This project helps businesses identify at-risk customers and take proactive measures to improve customer retention.

---

🚀 Features

* Predicts customer churn using a trained Machine Learning model.
* Simple and interactive web interface.
* Real-time predictions.
* Data preprocessing and feature encoding.
* Clean and easy-to-use UI.
* Fast prediction with a pre-trained model.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yalalapremchand/Customer-Churn-Prediction.git
```

### 2. Open the project

```bash
cd Customer-Churn-Prediction
```

### 3. Create a virtual environment (Optional)

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux / macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
